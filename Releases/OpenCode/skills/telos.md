# Telos Skill

Telos is your Life OS — a structured system for thinking about goals, priorities, beliefs, and personal strategy.

## Core Concepts

- **Telos** — your ultimate aim, what you're optimizing for at the deepest level
- **Goals** — specific outcomes that serve the telos
- **Projects** — active work items that move goals forward
- **Beliefs** — the mental models you operate with
- **Wisdom** — insights worth keeping permanently

## Workflows

### Life Review
```
telos review: assess current state of goals and priorities
```
1. List active goals
2. Assess progress on each
3. Identify blockers
4. Reprioritize based on telos alignment
5. Define next actions

### Goal Decomposition
```
telos decompose: [goal]
```
Break a goal into concrete milestones, each with a clear success criterion.

### Belief Audit
```
telos audit beliefs: [domain]
```
Surface the assumptions underlying your current approach. Challenge each one.

### Wisdom Capture
```
telos capture: [insight]
```
Store a hard-won insight in structured form so it survives across contexts.

## Memory

Store telos artifacts in `~/.config/opai/memory/learning/telos.md`.
Format: one insight per line with date, context, and the core claim.
