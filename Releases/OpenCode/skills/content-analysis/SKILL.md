# Content Analysis Skill

Extract maximum signal from any content format.

## Modes

### Extract Wisdom
Pull the most valuable, non-obvious insights from content. Focus on:
- Ideas that change how you think
- Frameworks and mental models
- Surprising facts or reversals
- Actionable takeaways

### Summarize
Compress content to its essential points without losing meaning.

### Deep Analysis
Go beyond summarization — critique the argument, identify assumptions, check the evidence, find gaps.

### Comparative Analysis
Compare two or more pieces of content on the same topic. Where do they agree? Disagree? What does each miss?

## Workflow

1. **Ingest** the content (URL, transcript, paste, or description)
2. **Identify** the core argument or thesis
3. **Extract** key claims, evidence, and insights
4. **Evaluate** quality of reasoning and evidence
5. **Synthesize** into the requested output format

## Output Format

```
CONTENT: [title/source]
CORE THESIS: [one sentence]
KEY INSIGHTS:
  1. [insight] — [why it matters]
  2. [insight] — [why it matters]
  ...
BEST QUOTE: "[quote]"
CRITIQUE: [what the content gets wrong or misses]
ACTIONABLE: [what to do with this]
```
